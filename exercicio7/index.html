algoritmo "ContagemInteligente"
var

    c,t: Inteiro

inicio

        Escreva ("Qual o número inicial da contagem? ")
        Leia (c)
        Escreva ("Qual o número final da contagem? ")
        Leia (t)
        Escreva ("Executando contagem ")
        
        Se (c >= t) entao
            EscrevaL ("regressiva...")
            Enquanto (c >= t) faca
                EscrevaL (c)
                c <- c - 1
            FimEnquanto

        Senao
            EscrevaL ("progressiva...")
            Enquanto (t >= c) faca
                EscrevaL (c)
                c <- c + 1
            FimEnquanto
        FimSe

A tag de citações é a tag <q> que significa quote (citação), e faz apenas duas aspas duplas, porém dá mais sentido (semântica) usar a tag que colocar as duas aspas duplas no texto manualmente.
